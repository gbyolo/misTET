**cooler module**

This module provides bbcode and emoticons support.<br>
Supported bbcode:<br>
`[b]text[/b]`<br>
`[i]italic[/i]`<br>
`[u]underlined[/u]`<br>
`[img]link img[/img]`<br>
`[url=link]text[/url]`<br>
`[url]link[/url]`<br>
`[code=language]code[/code]`<br>
`[code]code[/code]`<br>

Supported emoticons:<br>
:) :( ;) :D :P :S :O :lol :asd:
